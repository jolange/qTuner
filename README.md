##### Building
- **Prerequisites**: [Qt4](https://qt-project.org/)
- **Linux**: In the base-path:
<pre><code>$ qmake
$ make
$ ./bin/qTuner</pre></code>
- **Windows**
 * in Visual Studio Command prompt
<pre><code>$ qmake
$ nmake release</pre></code>
